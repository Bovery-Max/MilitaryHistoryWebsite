# ⚔️ Military History Archive

> An educational web archive exploring the full scope of military history — from ancient battlefields to modern warfare, legendary commanders, weapon evolution, and historical memorials.

---

## 📖 About the Project

**Military History Archive** is a static educational website designed for history enthusiasts, students, and researchers. It serves as a clean, accessible, and well-structured reference covering key pillars of military history across civilizations and eras.

The site is built entirely with vanilla **HTML**, **CSS**, and **JavaScript** — no frameworks, no dependencies — keeping it lightweight, fast, and easy to maintain.

---

## 🗂️ Content Areas

| Section | Description |
|---|---|
| ⚔️ **Ancient Battles** | Major conflicts from antiquity — Marathon, Thermopylae, Cannae and beyond |
| 🎖️ **Legendary Commanders** | Profiles of history's greatest military minds |
| 🔫 **Weapons & Technology** | The evolution of arms from bronze swords to gunpowder and beyond |
| 🪖 **Uniforms & Equipment** | How military dress and gear changed across centuries |
| 🏛️ **Memorials & Legacy** | Famous monuments, cemeteries, and the memory of war |

## 🚀 Getting Started

No installation or build step required. Simply clone the repository and open the project in your browser.

```bash
# Clone the repository
git clone https://github.com/your-username/military-history-archive.git

# Navigate into the project folder
cd military-history-archive

# Open in browser
open index.html
```

Or drag and drop `index.html` directly into any modern web browser.
